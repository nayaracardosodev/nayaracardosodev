### 🔭 O que estou construindo?

#### 🤖 **Automação de Monitoramento e Observabilidade - AWS** (Em desenvolvimento)
Arquitetura Serverless para monitorar oscilações de preço de insumos (Projeto Callebaut).
- **Arquitetura:** Python rodando em **AWS Lambda**, orquestrado via **EventBridge**.
- **Notificação & Resiliência:** Integração com **AWS SNS** para alertas em tempo real e monitoramento de saúde da aplicação via **Datadog**.
- **Lógica:** Consumo da API do Mercado Livre com filtragem inteligente e comparação de métricas de preço.

#### 🐍 **API RESTful de Gestão de Usuários (CRUD)**
Backend robusto focado em padrões de arquitetura e manutenibilidade.
- **Funcionalidades:** CRUD completo com persistência de dados e integração resiliente com a API **ViaCEP** (com controle de timeout).
- **Destaques:** Aplicação de **Clean Architecture** (divisão em camadas), higienização rigorosa de dados com **Regex** e tratamento global de exceções.
